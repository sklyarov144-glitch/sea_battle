# Пиратский Морской Бой: Карта Сокровищ

Playable-версия казуального пиратского морского боя на Phaser 3 для дальнейшей подготовки к публикации в Яндекс Играх.

## Запуск

```bash
npm install
npm run dev
```

Vite поднимет локальный сервер и покажет адрес в терминале.

## Что уже есть

- Поле 8x8 для игрока и бота.
- Автоматическая случайная расстановка флота: 4, 3, 2 и два корабля по 1 клетке.
- Скрытое поле врага и видимое поле игрока.
- Ход после попадания сохраняется, промах передает ход.
- Бот добивает найденные корабли: проверяет соседние клетки и продолжает линию, когда понимает направление.
- Событийные клетки на поле врага:
  - сундук;
  - бочка с порохом;
  - обломки с подсказкой;
  - водоворот;
  - мина.
- Способности игрока:
  - радар 3x3;
  - залп крестом;
  - торпеда по строке или колонке;
  - отмена режима прицеливания.
- Карта из 10 островов с прогрессией сложности.
- Победа, поражение, награды, сундук и заглушка rewarded-рекламы.
- Магазин с расходниками и скинами.
- Ежедневный сундук один раз в день.
- Сохранение прогресса в `localStorage`.
- Заглушка `YandexService` для будущего SDK Яндекс Игр.

## Архитектура

- `src/config/balanceConfig.js` — уровни, награды, сложность, магазин.
- `src/config/assetKeys.js` — ключи будущих ассетов и generated-текстур.
- `src/services/StorageService.js` — профиль игрока и сохранения.
- `src/services/YandexService.js` — безопасная заглушка SDK.
- `src/utils/boardUtils.js` — генерация досок, выстрелы, события, выбор выстрела ботом.
- `src/utils/effects.js` — визуальные эффекты Phaser Graphics/tweens.
- `src/ui` — переиспользуемые UI-компоненты.
- `src/scenes` — Boot, Preload, Menu, Map, Game, Result, Shop.
